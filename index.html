[ 02 ]

TESSERA란 무엇인가

신원의 증표, 다시 태어나다

고대 로마에서 테세라는 병사의 신원 증표이자, 아군과 적군을 가르는 암구호였습니다. 그 증표를, 지금 다시 만듭니다.

통신이 끊겨도 작동하는 NFC 플랫폼입니다. 무전원 태그 · 충전식 스캐너 · 보안 소프트웨어 — 스캔 한 번이 검증된 신원과 생명 직결 정보가 됩니다.

A

태그

무전원 NFC를 방수 HDPE에 봉인.

B

스캐너

충전식 견고한 리더기 · 에어갭 로컬 DB.

C

소프트웨어

보안·비식별 DB + 콘솔.

하나의 제품, 세 곳에서 동시에 — 병원 · 산업현장 · 군.

[ 03 ]

잠깐. 어디서 보신 것 같죠?

아이디어는 있었습니다. 아무도 보편화하지 못했을 뿐.
01

이 아이디어, 새롭지 않습니다

NFC 의료 디바이스는 이미 세상에 존재합니다.

02

그런데 사회는 그대로입니다

병원도, 산업현장도, 대한민국 육군도, 미군도 — 아무도 보편화시키지 못했습니다.

03

기존 의료용 NFC? 너무 비쌉니다

가격이 싼 경우엔 제대로 만들어지지 못했습니다. 합리적인 가격에, 의지할 만한 퀄리티로 — 테세라가 만듭니다.

이식형 NFC 칩침습적·고가 — 대량 보급 불가.
병원 RFID 손목밴드일회용·통신 의존 — 현장·오프라인에 부적합.
스마트 군번줄 시제품표준화 실패 및 보안 부재 — 채택되지 못함.

누군가는 제대로 해야 할 일입니다.
그래서, 우리가 합니다.

[ 04 ]

우리가 푸는 문제

몇 초가 결과를 바꾼다

신원은 아직도 ‘눈’으로 확인합니다 — 느리고, 위조 가능하고, 사람이 말 못 하면 아예 불가능합니다.

검증된 원천 문제

한미연합훈련에서 직접 목격

종이 TCCC 카드 대조로 처치가 지연되고, 통신 단절이 신원확인을 무너뜨렸습니다.

병원

체크인을 스캔으로

신분증·환자 정보를 암호화 스캔으로 — 더 정확하고, 더 빠르게.

응급

골든타임

말 못 하는 환자도 혈액형·알러지·병력을 즉시.

산업현장

사고 현장, 즉시

작업자 신원·위험정보·응급연락처 즉시. 중대재해처벌법으로 의무화.

보안 우선 설계

충전된 단말기에서 스캔 한 번 — 통신은 필요 없습니다. 병원에 실제로 배치되게 만드는 것은 보안입니다.

3
